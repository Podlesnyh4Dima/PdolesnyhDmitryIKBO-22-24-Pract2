## Общее описание

Данный проект представляет собой консольное приложение (**CLI**) для анализа и визуализации графа зависимостей пакетов.  
Программа строит граф зависимостей (включая транзитивные связи) с возможностью:
- фильтрации по подстроке,
- задания глубины анализа,
- генерации визуализаций в форматах **Graphviz** (PNG) и **Mermaid** (текстовая диаграмма).

Проект выполнен в рамках практической работы №2 по дисциплине  
**«Конфигурационное управление»**, **вариант №17**.

---

## Основные функции

### 1. Загрузка конфигурации

Конфигурация хранится в файле **`config.csv`** и содержит следующие параметры:

| Параметр | Описание |
|-----------|-----------|
| `package_name` | Имя анализируемого пакета |
| `repo_source` | URL репозитория или путь к файлу тестового графа |
| `repo_mode` | Режим работы |
| `package_version` | Версия пакета |
| `max_depth` | Максимальная глубина анализа |
| `filter_substring` | Подстрока для фильтрации пакетов |

При запуске программа выводит все параметры и проверяет их корректность.

---

### 2. Сбор данных о зависимостях

Поддерживается работа в двух режимах:
- **Реальный репозиторий npm** — получение данных через JSON API;
- **Тестовый JSON-файл** — локальное описание зависимостей для отладки.

Для каждого пакета извлекаются прямые зависимости (ключ — имя пакета, значение — версия).

---

### 3. Построение графа зависимостей

- Реализован алгоритм **DFS (обход в глубину)** без рекурсии.  
- Учитываются ограничения по глубине (`max_depth`) и фильтрация по подстроке.  
- Корректно обрабатываются циклические зависимости.  
- Поддерживается **режим тестирования** с локальным JSON-графом.

---

### 4. Дополнительные операции

- Генерация **порядка загрузки зависимостей** (топологическая сортировка);
- Проверка на циклы и вывод ошибок при их обнаружении;
- Вывод полного списка зависимостей в консоль в виде словаря.

---

### 5. Визуализация

- Генерация текстового представления графа в формате **Mermaid**;
- Визуализация графа через **Graphviz**, результат сохраняется в `dependency_graph.png`;
- Вершины и рёбра визуализируются с подписями и цветовой заливкой.
